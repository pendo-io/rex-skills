# Unverified skills

This folder is the landing zone for new skills contributed by anyone at Pendo.

## How to contribute

1. Fork (or create a branch if you have push access).
2. Drop your skill in a new subfolder here: `unverified/<your-skill-name>/SKILL.md` (use `templates/SKILL_TEMPLATE.md` as a starting point).
3. Open a PR.

Anything in `unverified/` is treated as community-contributed and not yet reviewed by the Glass team. Once a skill is reviewed and signed off, it gets moved out of `unverified/` into the appropriate kit folder (`swe/`, `sales/`, `csm/`, etc.).

## Why a separate folder?

So Glass can surface community skills distinctly from the curated, reviewed ones — and so contributors can ship without waiting on review.
